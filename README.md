# 🥋 Jiu-Jitsu Check-In
## 📌 Sobre o Projeto

* O Jiu-Jitsu Check-In é um sistema web para controle de presença e frequência de atletas no tatame em tempo real. Desenvolvido com foco em performance, código limpo e manipulação precisa do DOM utilizando JavaScript Vanilla.
## 🚀 Funcionalidades

* Cadastro de Atletas: Registro de novos praticantes informando nome e graduação (faixa).

* Painel de Treino (Lista Dinâmica): Visualização clara de quem está ativamente no tatame.

* Check-Out (Exclusão): Saída do atleta do tatame com atualização imediata da lista.

* Validação de Segurança: Sistema inteligente que impede registros com dados em branco.

## 🧠 Aprendizados e Desafios (Sessão de Treino)

Este projeto foi desenvolvido utilizando a metodologia ativa de resolução de problemas, refinando conceitos fundamentais de:

    Fluxo de execução e validações com condicionais (if/return).

    Delegação de eventos com event.target e classList.contains.

    Ligação de dados (data-binding) com atributos customizados data-*.

    Métodos avançados de manipulação de arrays (splice e findIndex).
